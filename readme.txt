The Pearl Jam Taglines plugin allows you to randomize your self-hosted WordPress blog's tagline with one of 50 classic Pearl Jam lyrics.

== Description ==

The Pearl Jam Taglines plugin allows you to randomize your self-hosted WordPress blog's tagline with one of 50 classic Pearl Jam lyrics. Because let's face it, Eddie Vedder is much better at writing clever things than you are.

The tagline will be updated on each page load (in the next version, you'll be able to customize this)

Please submit any bug reports, support questions, or requests <a href="http://anthonybubel.com/contact/">here</a>. Suggestions/requests for the addition of any of your personal favorite lyrics are always welcome! Please note that requests are not guaranteed to be added.

== Installation ==

1. Upload `pearl-jam.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins -> Installed' menu in your dashboard.

== Frequently Asked Questions ==

= How often does the tagline update? =

On each page load. The next version will allow you to customize this.

= Which lyrics are included? =

Just keep refreshing your blog. :)

If you really need to know, simply open the `pearl-jam.php` file in a text editor, or open it via 'Plugins -> Editor' in your dashboard and view the list there.
